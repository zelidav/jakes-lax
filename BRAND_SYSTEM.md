## 2) Design System

**Layout rules**
- Back graphic centered, 10–12 in wide
- Front = small left-chest hit
- One clear focal moment
- Humor reads instantly — no clutter

**Visual style**
- Thick outlines, screen-print friendly
- High contrast, 2–4 colors max
- Negative space background
- Slight grit / texture (not overdone)

**Character direction**
- College lacrosse player, helmet always on (face visible through cage)
- Expressive reactions — embarrassed, cocky, frustrated
- Slightly exaggerated, still athletic
- Reuse the same character across designs

**Color system — pick one per drop**
- Black / cream / red
- Gray / navy / white
- Off-white / vintage tones

**Typography**
- Bold collegiate / varsity fonts, slight distress
- Strong hierarchy: top line hook, bottom line punchline
- Slight arc works for back graphics

---

## 3) Midjourney Art Rules

Always generate *without text* first.

**Base prompt add-on**
```
bold t-shirt graphic, centered composition, thick outlines, high contrast,
limited colors, screen print friendly, vector style, minimal background
```

**Settings**
- `--style raw`
- `--ar 1:1`
- `--s 400–700` (clean)
- `--s 700–900` (funnier / more expressive)

**Process:** generate → upscale best → variations → add text in Illustrator/Photoshop.

## 5) Drop Structure

**Drop = 5–8 designs max**
- 2 hero
- 3 funny
- 2 clean/premium
- 1 wildcard

**Cadence:** monthly or seasonal; build anticipation via Instagram.

---

## 6) What Makes These Sell

- Simplicity wins
- Readable from 10 feet
- One joke per shirt
- Consistent character builds recognition
- Mix funny + clean so people will wear them
